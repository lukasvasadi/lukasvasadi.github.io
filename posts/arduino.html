Introduction

Arduino is an opensource platform for microcontroller boards.

Code Structure

int led = 13;

void setup() {

// set led pin as output

pinMode(led, OUTPUT);

}

void loop() {

// toggle led on and off

digitalWrite(led, HIGH);

delay(1000);

digitalWrite(led, LOW);

delay(1000);

}
