Introduction

This tutorial will guide you through configuring a local development environment for popular tools, such as Python, Node.js and Git, as well as system package managers.

Windows

Chocolatey

Chocolatey is probably the best general package manager for Windows. With one command, this tool bypasses the process of downloading executable installation wizards.

To install Chocolatey, run the following commands in an Administrative PowerShell session:

Set-ExecutionPolicy Bypass -Scope Process -Force

[System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072

iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

Pyenv

Install pyenv with choco:

choco install pyenv-win

Confirm pyenv installation:

pyenv --version

Install a version of Python:

3.10.0 as an example

pyenv install 3.10.0

Set the global Python version:

pyenv global 3.10.0

Confirm that python and pip point to the correct executables:

where python

C:\Users\user\.pyenv\shims\python

where pip

C:\Users\user\.pyenv\shims\pip

Mac

Homebrew

Similar to Chocolatey, Homebrew probably ranks as the top tool for general package management in MacOS.

To install Homebrew:

curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh | zsh"

Pyenv

To install pyenv, first install the package dependencies:

brew install openssl readline sqlite3 xz zlib

Then execute the pyenv installer:

curl https://pyenv.run | zsh

Update the PATH environment variable:

echo 'export PATH="$HOME/.pyenv/bin:$PATH"' >> ~/.bashrc

echo 'export PATH="$HOME/.pyenv/shims:$PATH"' >> ~/.bashrc

echo 'eval "$(pyenv init -)"' >> ~/.bashrc

echo 'eval "$(pyenv virtualenv-init -)"' >> ~/.bashrc

Refresh the environment:

exec $SHELL

Confirm pyenv installation:

pyenv --version

Install a version of Python:

3.10.0 as an example

pyenv install 3.10.0

Set the global Python version:

pyenv global 3.10.0

Confirm that python and pip point to the correct executables:

which python

/Users/user/.pyenv/shims/python

which pip

/Users/user/.pyenv/shims/pip
