Introduction

This guide assumes a basic knowledge of bash and the physical layout of a Raspberry Pi.

Remote access

SSH

Assuming the Pi is running Raspberry Pi OS, enable SSH in system settings and reboot. Find the IP address with ifconfig and then use the ssh utility on the remote machine (remember that "pi" and "raspberry" are the default username and password, respectively):

ssh pi@xxx.xx.xx.xx

Updates

Package updates are pushed periodically, so it is recommended to frequently run the following commands to keep your system current. To update the system package list and then upgrade all installed packages:

sudo apt-get update

sudo apt-get dist-upgrade
